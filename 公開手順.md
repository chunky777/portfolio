# ポートフォリオを公開URLにする手順

このフォルダ（`ポートフォリオ`）をそのまま使って、無料で公開URLを取得できます。

---

## 方法1: Netlify Drop（いちばん簡単・おすすめ）

**アカウント不要で即URLがもらえます。**

1. ブラウザで **https://app.netlify.com/drop** を開く
2. **「ポートフォリオ」フォルダ全体**をドラッグ＆ドロップする
3. 数秒で公開URLが表示されます（例: `https://xxxxx.netlify.app`）
4. そのURLをコピーして共有すればOK

※ アカウントを作ると、同じURLを維持したりカスタムドメインを設定できます。

---

## 方法2: GitHub Pages（無料・URLを維持したい場合）

**GitHubアカウントがある人向け。`あなたのユーザー名.github.io/リポジトリ名` のようなURLになります。**

1. **GitHub** で新しいリポジトリを作成（例: `portfolio`）
2. この **「ポートフォリオ」フォルダの中身**（`index.html` と `portfolio_2.html`）をリポジトリにアップロード
   - ブラウザから「Add file」→「Upload files」で `index.html` をアップロードしてもOK
3. リポジトリの **Settings** → **Pages** を開く
4. **Source** で「Deploy from a branch」を選び、**Branch** を `main`、フォルダを `/ (root)` にして **Save**
5. 数分後に次のURLで表示されます:
   - `https://あなたのユーザー名.github.io/portfolio/`  
   （リポジトリ名が `portfolio` の場合）
   - リポジトリ名を `あなたのユーザー名.github.io` にすると  
     `https://あなたのユーザー名.github.io/` でトップに表示されます

---

## 方法3: Vercel（無料・簡単）

1. **https://vercel.com** にアクセスし、GitHubでログイン
2. **Add New** → **Project** で、このフォルダをGitで連携するか、**Import** でフォルダをアップロード
3. デプロイ後、`https://xxxxx.vercel.app` のようなURLが発行されます

---

## 補足

- **index.html** を用意してあるので、どの方法でもトップのURL（`https://xxx.com/`）でポートフォリオが表示されます。
- **portfolio_2.html** はそのまま残してあるので、`https://xxx.com/portfolio_2.html` でも開けます。
- 内容を変えたら、同じ手順でもう一度アップロード（またはプッシュ）すると更新されます。

質問があれば、どの方法でやるか教えてください。
